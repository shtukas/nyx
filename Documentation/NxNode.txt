NxNodes are the nodes of the Nyx network.

Each orbital is a collection of records of the form (key: text, collection: text, data: blob)

Used as a key value store, we consider records of the form (key, null, data). Used to manipulate sets we consider records of the form (null, collection-name, data)

-------------------------------------------------------------------
key                 |                    | data
uuid
unixtime                                                            float
datetime
description
coredataref
taxonomy

key                 | collection         | data
linked:<linkeduuid> | linked             | <linkeduuid>
<uuid>              | note               | NxNote json encoded
<uuid>              | coredata-reference | CoreDataRef json encoded
-------------------------------------------------------------------

Unless otherwise specified the data of the key / value store are strings. unxitime is a float.

Orbital file names follow a strict structure

<main>.nyxnode

The ~/Galaxy/DataHub/Nyx/Index directory contains individual files, each named after and containing an orbital uuid.
